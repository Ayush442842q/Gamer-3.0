# Candy Crush Saga

A beautiful, premium candy matching puzzle game built with HTML, CSS, and Vanilla JavaScript.

## Features
- Classic match-3 gameplay with cascades
- Special candies (Striped, Wrapped, Color Bomb)
- Dynamic CSS themes
- Synthesized Sound Effects (Web Audio API)
- Boosters / Power-ups
- High Score Leaderboard

## Development
To run this game:
1. Simply open the `index.html` file in any modern web browser.
2. Or run a local dev server (e.g., using Live Server or `python -m http.server`).
